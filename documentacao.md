# Documentação do Projeto One Time Pay

## Estrutura do Projeto

Este é um projeto de landing page para o One Time Pay, um sistema de checkout transparente.

## Índice de Componentes e Funções

### HTML
1. **index.html** - Página principal da landing page
2. **privacidade.html** - Página de política de privacidade

### JavaScript
3. **openContactForm()** - Função que abre o modal de contato
4. **closeContactForm()** - Função que fecha o modal de contato
5. **openPrivacyPolicy()** - Função que abre o popup de política de privacidade
6. **closePrivacyPolicy()** - Função que fecha o popup de política de privacidade
7. **openTermsOfUse()** - Função que abre o popup de termos de uso
8. **closeTermsOfUse()** - Função que fecha o popup de termos de uso
9. **AOS.init()** - Inicialização das animações de scroll

### CSS
10. **styles.css** - Estilos principais da landing page
11. **animations.css** - Animações customizadas
12. **privacy-popup.css** - Estilos do popup de privacidade
13. **terms-popup.css** - Estilos do popup de termos de uso

### Elementos Visuais
14. **Hero Section** - Seção principal com título e CTA
15. **Benefits Section** - Seção de benefícios do checkout transparente
16. **Stats Section** - Seção com estatísticas de performance
17. **Advantages Section** - Comparação com/sem checkout transparente
18. **Integration Section** - Passo a passo de integração
19. **FAQ Section** - Perguntas frequentes com accordion
20. **Contact Modal** - Modal de formulário de contato
21. **Privacy Popup** - Popup com política de privacidade
22. **Terms Popup** - Popup com termos de uso
